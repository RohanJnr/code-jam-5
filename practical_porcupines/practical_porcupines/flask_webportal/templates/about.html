About Practical Porcupine

What does it do?

Practical Porcupine is a project hacked together for the 5th Python code jam, with the theme of Global Warming. This web-portal connects to an api (that you are most likely running on your computer right now (If you are, it may be here)! It gets 2 dates, as seen on the home page and calculates the difference between the GMWL (Global Mean Water Level) in mm.

To save database space and show off, we have a large dataset that has been interpolated to give you precise times of the GMWL. There is also a simple prediction algorithm that was going to be machine learning but there was not enough time to test unforunatly.
